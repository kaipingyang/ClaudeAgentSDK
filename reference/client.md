# ClaudeSDKClient — Bidirectional Interactive Client

R6 class for interactive, stateful conversations with Claude Code.
Mirrors `client.py` from the Python SDK.

For simple one-shot queries use
[`claude_run()`](https://kaipingyang.github.io/ClaudeAgentSDK/reference/claude_run.md)
or
[`claude_query()`](https://kaipingyang.github.io/ClaudeAgentSDK/reference/claude_query.md).
